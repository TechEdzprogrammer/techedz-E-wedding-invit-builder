# TechEdZ E-wedding invitation builder Landing Page

This is a static landing page for showcasing your four wedding invitation themes.

## Edit the live links
Open `theme-links.js` and replace the placeholder Netlify URLs:
- signature
- personal
- booklet
- classic

## Deploy
1. Push these files to a GitHub repository.
2. Import the repo into Netlify.
3. Publish as a static site.

No build step is required.
